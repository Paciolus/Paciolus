## Overview

This log records completion of mandatory security awareness training for all Paciolus team members. Training is required:
- **On hire:** All applicable modules completed within 30 days of start date
- **Annual refresh:** All applicable modules re-completed by January 31 each year

**Curriculum reference:** `docs/08-internal/security-training-curriculum-2026.md`

**Module guide:**

| ID | Module | Duration | Required For |
|----|--------|----------|-------------|
| M1 | OWASP Top 10 | ~1 hour | All |
| M2 | Secure Coding — Python + TypeScript | ~1 hour | Engineers |
| M3 | Incident Response Roles + Escalation | ~30 min | All |
| M4 | Access Control + Least Privilege | ~30 min | All |
| M5 | Social Engineering + Phishing | ~30 min | All |
